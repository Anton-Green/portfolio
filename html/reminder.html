Reminder and history solutions explained:

Php script takes note when user was last time logged in, on document load js func checking last login of all users, if last login was more than a week ago, then func is sending a reminder to user e-mail, after that history will be deleted.

In that way, we can create balance between spaming and reminding user. Also func that used for sending e-mail can be used for sending any other info by changing func's input.
